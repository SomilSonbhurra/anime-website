# Anime Website 🎥
Full-stack anime project using React + Node.js.

## 📦 Features
- Anime search, categories, trailers
- Login system
- Episode streaming

## 📂 Structure
- `client/`: React frontend
- `server/`: Express backend

## 🚀 Deployment

- Frontend on [Vercel](https://vercel.com)
- Backend on [Render](https://render.com) or Railway

## 🛠 Setup

```bash
cd client
npm install
npm start

cd server
npm install
node server.js

-----
